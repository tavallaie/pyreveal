## Quick example

```python
from pyreveal import PyReveal, Slide

presentation = PyReveal(title="Hello", theme="dracula", transition="fade")
presentation.configure(hash=True, progress=True)

intro = Slide(content="<h1>Welcome</h1>")
intro.add_vertical_slide(Slide(content="<p>Built with PyReveal</p>"))
presentation.add_slide(intro)

presentation.save_to_file("deck.html")
```

The interactive demo above is generated by `scripts/build_docs_demo.py` using the same API.
